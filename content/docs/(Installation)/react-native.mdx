## Prerequisites

- React Native CLI
- Node.js
- npm or yarn

## Installation

<Tabs groupId='package-manager' persist items={['npm', 'pnpm', 'yarn', 'bun']}>

```bash tab="npm"
npm i @okto-sdk/react-native-sdk
```

```bash tab="pnpm"
pnpm i @okto-sdk/react-native-sdk
```

```bash tab="yarn"
yarn add @okto-sdk/react-native-sdk
```

```bash tab="bun"
bun add @okto-sdk/react-native-sdk
```

</Tabs>

## Configuration

After installation, you need to configure the Okto SDK. This involves setting up your environment variables and initializing the SDK with your project's details.

```typescript heading="App.tsx"
// App.tsx
import React from "react";
import { OktoProvider } from "@okto/react-native-sdk";
const App = () => {
  return (
    <OktoProvider
      config={{
        apiKey: "your_api_key_here",
        environment: "testnet", // or 'mainnet'
      }}
    >
      <YourApp />
    </OktoProvider>
  );
};
export default App;
```
