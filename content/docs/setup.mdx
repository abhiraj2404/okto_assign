## Setting Up the Okto SDK

Welcome to the setup guide for the Okto SDK! This page will walk you through the installation and configuration process, so you can start building powerful blockchain applications with ease.

---

## Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** (v16 or higher)
- **npm** or **yarn** (for package management)
- A code editor (e.g., [VS Code](https://code.visualstudio.com/))

---

## Installation

To install the Okto SDK, run the following command in your terminal:

<Tabs groupId='package-manager' persist items={['npm', 'pnpm', 'yarn', 'bun']}>

```bash tab="npm"
npm i okto-sdk -D
```

```bash tab="pnpm"
pnpm i okto-sdk -D
```

```bash tab="yarn"
yarn add okto-sdk -D
```

```bash tab="bun"
bun add okto-sdk -D
```

</Tabs>
